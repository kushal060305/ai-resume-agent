# 🚀 AI Resume Tailoring Agent

An AI-powered ATS system that analyzes resumes, detects skill gaps, and generates optimized resumes.

## 🔥 Features
- ATS Score Calculation
- Skill Gap Detection
- Resume Optimization
- PDF Upload Support
- Interactive Web UI (Streamlit)

## 🛠️ Tech Stack
- Python
- NLP (spaCy, Sentence Transformers)
- Streamlit UI

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run ui.py
